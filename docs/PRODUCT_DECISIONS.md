# Product decisions

## Build as separate product

B20 Watcher should not wait for AgentVault+. It should be built as an independent public dashboard and API.

## MVP scope

Ship only:

- B20 token discovery
- token detail page
- issuer-control risk score
- role/policy/supply/pause event timeline
- JSON API

Do not ship yet:

- wallet custody
- trading
- buy/sell signals
- full AI agent automation
- legal/compliance advice
- custom indexer
- full paid billing system

## Monetization path

Version 1:

- free public scanner
- shareable token report URL

Version 2:

- paid detailed reports
- alerts
- API key access

Version 3:

- x402 pay-per-scan
- AgentVault+ integration
- risk oracle for agents
